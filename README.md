# 小红书采集工具爬虫

node版本: v16.20.0

执行 `npm install` 安装依赖

执行 `npm run start` 开始爬取网站数据

### 功能介绍

小红书数据爬虫项目，每日下午5点从云盘拉取待爬取的数据名单，根据名单执行爬虫任务将当日爬取的数据上传到云盘，并发送钉钉消息提醒。


### 待爬取表格必须有的表头如下

| ID    | NoteNumber | author |           link            |
|:------|-----------:|:------:|:-------------------------:|
| 编号001 |      自产001 |  芋头啊   | http://xhslink.com/LnIqKr |
| 编号002 |      自产002 |  小芋头   | http://xhslink.com/teVZxr |
